About Castro

Castro is a compressible radiation hydrodynamics code for modeling astrophysical flows. It is built upon the BoxLib adaptive mesh refinement library and incorporates an unsplit Godunov scheme for hydrodynamics, gravity via a Poisson solver or multipole expansion, reaction networks, and a choice of equations of state.

To get started, clone Castro and try running the standard Sedov problem: 

$ cd Castro/Exec/Sedov
$ make DIM=2 COMP=gcc FCOMP=gfortran USE_MPI=FALSE
$ ./Castro2d.Linux.gcc.gfortran.ex inputs.2d.sph_in_cylcoords

Full details on the compilation options in the GNUmakefile can be found in the Castro User's Guide.

Check out the Castro gallery of simulations to see some of the applications Castro has run.

Working with the Code

There are several modules available that extend the core Castro functionality:

CastroRadiation provides the multigroup flux-limited diffusion module for Castro.

wdmerger provides the ability to model binary white dwarf systems.
