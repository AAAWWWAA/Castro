subroutine ca_initdata(level level, time time, lo lo, hi hi, nvar nvar, state state, state_l1 state_l1, state_l2 state_l2, state_h1 state_h1, state_h2 state_h2, dx dx, xlo xlo, xhi xhi)¶

This routine is called at problem setup time and is used to initialize data on each grid.

INPUTS/OUTPUTS:

Note
all arrays have one cell of ghost zones surrounding the grid interior. Values in these cells need not be set here.

level => amr level of grid time => time at which to init data lo,hi => index limits of grid interior (cell centered) nvar => number of state components. state <= scalar array dx => cell size xlo, xhi => physical locations of lower left and upper right hand corner of grid. (does not include ghost region).
